1 Introduction

Webinfo is a documentation server for Web and Desktop that uses Texinfo model for its documents.

• Why Texinfo?  

Up: Introduction   [Contents]

1.1 Why Texinfo?

  • A good document model formed by trees of nodes.
  • Access to the document model via XML serialization (--xml option in texi2any).
  • Support for definitions that appear in programming languages, like functions, variables, types, and more.
  • Lots of documentation already written in Texinfo format.

3 Usage

• Command line  

  [Contents]

Webinfo as standalone Texinfo reader

Webinfo can be used as a standalone Texinfo reader.

After building and installing webinfo command line executable, just run it on a Texinfo file: 

webinfo - WebInfo - A Texinfo documents reader for desktop and Web.

USAGE: webinfo [OPTIONS] TEXINFO-FILE

Open TEXINFO-FILE with WebInfo reader.

Options:
  --version             display version information and exit
  -h, --help            display help information and exit
  --debug               show errors
  --lenient             ignore unrecognized Texinfo commands
